# Aperçu des pratiques

Ce dossier regroupe toutes les pratiques d’IndabaX Guinée 2025. Chaque session possède :

- `README.md` : résumé, objectifs, prérequis, déroulé.
- `notebooks/` : notebooks ou scripts.
- `slides/` : présentations.
- `resources/` : données ou liens utiles.

- [`health-information-systems-data-integration-cloud-hosting`](health-information-systems-data-integration-cloud-hosting/) – Systèmes d’information santé : intégration et hébergement cloud
- [`fondamentaux-du-deep-learning`](fondamentaux-du-deep-learning/) – Fondamentaux du deep learning
- [`ia-generative-innovation-locale`](ia-generative-innovation-locale/) – IA générative : de la théorie à l’innovation locale
- [`pipeline-aux-modeles-ecosysteme-data`](pipeline-aux-modeles-ecosysteme-data/) – Du pipeline aux modèles : écosystème data complet
