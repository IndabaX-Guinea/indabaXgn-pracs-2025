# Guide de contribution

Merci de préparer une pratique pour **IndabaX Guinée 2025**. Suivez les étapes ci-dessous pour garder le dépôt clair et léger.

## Étapes

1. **Créer une branche** depuis `main` (ex. `feature/atelier-nlp`).
2. **Mettre à jour votre dossier** dans `practicals/` :
   - compléter `README.md` (résumé, objectifs, prérequis, installation, déroulé, contacts) ;
   - déposer notebooks dans `notebooks/`, slides dans `slides/`, ressources dans `resources/` ;
   - ajouter `requirements.txt`, `environment.yml` ou autre fichier d’environnement si nécessaire.
3. **Tester** la procédure d’installation sur un environnement propre.
4. **Ouvrir une pull request** vers `main` avec un court résumé.

## Données

- Évitez les fichiers >100 MB ; contactez l’équipe si vous avez besoin de Git LFS ou d’un autre hébergement.
- Supprimez toute donnée sensible ; privilégiez des jeux publics ou synthétiques.

## Style

- Utilisez des titres clairs dans les notebooks et des commentaires courts.
- Restez dans la même langue (ici français) pour l’ensemble du dossier.
