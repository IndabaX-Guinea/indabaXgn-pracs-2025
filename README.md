# IndabaX Guinée 2025 – Pratiques

Répertoire central pour les sessions pratiques d’**IndabaX Guinée 2025**. Chaque intervenant y dépose ses supports pour faciliter la préparation des participantes et participants.

## Structure

- `practicals/` : dossiers de chaque session (README, notebooks, slides, ressources).
- `LICENSE` : licence commune des contenus partagés.

La liste détaillée des sessions se trouve dans `practicals/README.md`.

## Intervenants

1. Créez une branche dédiée (ex. `feature/rag`).
2. Complétez le dossier de votre session : README, notebooks, slides, ressources, fichiers d’environnement.
3. Vérifiez vos instructions sur une machine propre.
4. Ouvrez une pull request vers `main` pour validation.

## Participants

- Consultez `practicals/` pour retrouver votre atelier.
- Suivez les instructions d’installation avant le jour J.
- Revenez après l’événement pour les mises à jour éventuelles.

## Démarrage rapide

```bash
git clone git@github.com:IndabaX-Guinea/indabaXgn-pracs-2025.git
cd indabaXgn-pracs-2025
python3 -m venv .venv
source .venv/bin/activate  # fish : source .venv/bin/activate.fish
```
